# Cursor Agent Documentation

This folder contains documentation and resources for AI agent collaboration on the Rehab Budget Pro project.

## Folder Structure

```
.cursor/
├── rules/           # Development rules and guidelines
├── tools/           # Agent tools and utilities
├── docs/            # Technical specifications and design docs
└── notes/           # Project notes and checklists
```

## Quick Start for Agents

1. **Read `notes/agentnotes.md`** - Critical project context and patterns
2. **Check `notes/project_checklist.md`** - Current tasks and progress
3. **Review `notes/notebook.md`** - Important learnings and gotchas
4. **Follow rules in `rules/`** - Development guidelines

## Documentation Files

### Notes Folder
- **agentnotes.md** - Essential information for agent sessions
- **project_checklist.md** - Task tracking and progress
- **notebook.md** - Findings, learnings, and useful tidbits

### Rules Folder
- **agent_rules.md** - Agent collaboration guidelines
- **nextjs_rules.md** - Next.js and App Router patterns
- **react_rules.md** - React component patterns
- **supabase_rules.md** - Database and Supabase patterns

### Docs Folder
- **architecture-overview.md** - System architecture and design

## Maintenance

- Update documentation frequently during development
- Keep project_checklist.md current with task status
- Document interesting findings in notebook.md
- Add new rules as patterns emerge

## User Commands

- **"doc it"** - Update all documentation with current state
- **"Start me up"** - Review all documentation to get up to speed
